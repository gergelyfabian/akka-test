akka-test
=========

Testing Akka.

Compile the project with:
------------------

    $ mvn package

Run the project with:
------------------

    $ scala -cp target/akka-test-1.0-SNAPSHOT.jar hu.ebratanki.App

or

    $ java -cp target/akka-test-1.0-SNAPSHOT-jar-with-dependencies.jar hu.ebratanki.App
